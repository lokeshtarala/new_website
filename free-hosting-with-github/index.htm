Free Hosting With Github

by 1laptop | Aug 9, 2020 | Github pages | 0 comments

Free hosting with the Github

The journey of a website developer is full of mysteries and fun.

Paying 3 to 4 hundred rupees for a month may not be affordable to all, and that should not stop you from becoming what you want to.

I want to do digital marketing, blogging, content marketing, I want to learn SEO practically by launching my website and apply my strategies on it, but I need a website.

So you are welcome to the Github community where you can make GitHub pages and publish them.

Here is a step by step process of starting your journey of a web developer with Github.

Before starting, there are some limitations to the Github pages, but it won’t be a problem if you are beginning or just blogging only. Here are the restrictions:

1.GitHub Page’s source repositories have a recommended limit of 1GB. 

2. Published GitHub Pages sites may be no larger than 1 GB.

3. GitHub Pages sites have a soft bandwidth limit of 100GB per month.

As a beginner or blogger, 1Gb for one account is WOoow. It’s so much.

Step 1:

Create the Github account here Github sign up 

Step 2:

 

 

Now create a new repository by clicking new on the button shown at top left beside repositories heading

 

 

 

Name the repository and set it to public so that every one access your website after made.

Check the box that says Initialize this repository with a README.

and then click on create repository (green button).

Hurray you have just created your repository.

Step 3:

You will be redirected to your repository after creating it , you can access from repository section also after you login into your github account.

Click on Add file and under that choose create new file (or) you can also upload you css and html files from your computer

create a html file for that you have to name it as name.html

and Place the html code inside the block named edit new file like simply as shown.

Hello world

And then click on Commit new file.

Now you are half way done.

 

Step 4:

Now click on settings on top menu after opening the repository created by you.

 

 

 

 

Scroll down till you find Github pages.

 

 

 

Select the branch under which you have created new file (if not created and selected any branch default branch is master branch)

Keep Folder as it is i.e., /(root) 

And click on save.

Congo you have created your gihub page

You can find its link in settings in Github pages.
